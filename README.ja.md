# ライブWeb証拠エージェント

[English README](README.md)

## 一言でいうと

AIでライブWeb調査をするチームのために、現在のページを主張台帳に変え、自信満々の回答をレビュー可能な証拠に変えるツールです。

## 誰のためのものか

創業者、投資家、プロダクトチーム、運用担当者など、AIにWeb調査を任せたいが、根拠と承認状態を失いたくない人のためのワークフローです。

## 何が困りごとか

AIはWeb調査の答えを速く出せます。しかし、どの主張がどの情報源から来たのか、古いのか、弱いのか、まだ言ってはいけないのかが見えないと、チームは安心して使えません。

## どう解決するか

- 情報源カードを作ります。
- 主張を抽出します。
- 信頼度と判断状態を表示します。
- 危ない主張をブロックします。
- 人間へ引き継ぐパケットを作ります。
- 承認用の回答を出力します。

## 日本語対応

画面右上の `日本語` ボタンで、日本語UIに切り替えられます。作者が日本人であることも含め、AI時代のプロダクトは最初から多言語で読めるべきだという思想を反映しています。

## 主張の境界

公開静的版は、証拠ワークフローを示すものです。まだ実Nimble API利用、ライブ抽出、自動で真実確認済みの回答は主張しません。

## 検証

```bash
npm run verify
```

検証では、情報源カード、主張台帳、ブロック主張、人間引き継ぎ、日本語UI切替、デモ動画、no-secrets確認をまとめて確認します。
